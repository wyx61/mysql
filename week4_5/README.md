### Week4 功能

1. 使用 React 创建前端页面。

2. 使用 Flask 创建后端接口。

3. 前端包含三个输入框和两个按钮。

4. 第一个按钮通过 GET 请求，将第一个输入框中的内容作为 URL 参数发送给后端。

5. 后端接收参数后返回：`参数是 xxx`。

6. 第二个按钮通过 POST 请求，将第二个输入框作为 body 参数，将第三个输入框作为 URL param 参数发送给后端。

7. 后端接收后返回：`body中的参数是 xxx，param中的参数是 xxx`。

### Week5 功能

1. 使用 MySQL 创建数据库和数据表。

2. Flask 后端使用 pymysql 连接 MySQL 数据库。

3. 后端提供 `/api/sales` 和 `/api/categories` 两个接口。

4. `/api/sales` 查询每月销售额数据，用于柱状图和折线图展示。

5. `/api/categories` 查询分类占比数据，用于饼图展示。

6. React 前端通过 fetch 请求 Flask 后端接口获取 JSON 数据。

7. 前端使用 echarts-for-react 绘制柱状图、折线图和饼图。

8. 前端使用 Ant Design 的 Menu 组件实现不同图表之间的切换。
